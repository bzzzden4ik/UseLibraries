## What it can do now?

There is only 1 container:

### List

Simple std::vector implementation with extended functional that can help become your project simpler and cleaner.

Now it can:
* append(const T& value) - add element at the end of data[]
* pop() - return last element and delete it from data[]
* print() - print elemnts of non-composite type data[] (now in progress)
* size() - return length of data[]
* cap() - return capacity of data[]
* List1 + List2 - key feature (like in Python and other languages)
* some overloaded operators that make usage easier
